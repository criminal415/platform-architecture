# Architecture Decision Records (ADR)

이 폴더에는 플랫폼 전역의 중요한 아키텍처 결정을 기록한다.

- 파일명 형식: `NNNN-title.md`
- 상태(Status): Proposed → Accepted → Superseded
- 범위:
  - 플랫폼 전역에 해당하는 결정만 기록
  - 서비스/도메인별 결정은 해당 디렉토리에 별도 기록
